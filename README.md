# Portfolio

Modern portfolio website built with React, Vite, and Framer Motion.

## Live Site

https://khalandars.github.io/port/

## Features

- Animated particle background
- Liquid button animations
- 3D tilt profile card
- Interactive skills showcase
- Projects gallery
- Contact section with social links
- Downloadable resume

## Development

```bash
npm install
npm run dev
```

## Deployment

Automatically deploys to GitHub Pages on push to master branch.
